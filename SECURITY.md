# Security Policy

## Supported state

This project is in MVP foundation phase. Security-sensitive changes should be made conservatively and reviewed.

## Reporting a vulnerability

Please report vulnerabilities privately to the maintainers.

When reporting, include:
- affected files/package
- reproduction steps
- expected vs actual behavior
- potential impact

## Security design expectations

- default-deny source policy
- no unrestricted external retrieval
- deterministic validation and anomaly behavior
- no personal data logging in core flows
